# SuitUp (ارتداء بدلة) 🤵

تطبيق فلاتر عصري وأنيق يتيح للمستخدمين استعراض مجموعة من البدلات الفاخرة وتجربة ارتدائها افتراضياً (Virtual Try-On) باستخدام محاكاة الكاميرا.

## المميزات ✨
- **تصفح البدلات**: استعراض أحدث تصاميم البدلات بأسلوب عرض جذاب.
- **تفاصيل البدلة**: عرض التفاصيل الدقيقة للبدلة مع الصور والأسعار والتقييمات.
- **تجربة الارتداء الافتراضية**: واجهة تحاكي الكاميرا الأمامية لتجربة قياس البدلة افتراضياً قبل شرائها.
- **تصميم متجاوب**: يدعم جميع الشاشات (الهواتف، الأجهزة اللوحية، والويب).
- **واجهة باللغة العربية**: دعم كامل للاتجاه من اليمين إلى اليسار (RTL).

## التقنيات المستخدمة 🛠️
- Flutter & Dart
- Material 3 Design
- Responsive Layouts
- Navigation & Routing

## كيفية التشغيل 🚀
1. قم بتثبيت التبعيات:
   ```bash
   flutter pub get
   ```
2. شغل التطبيق:
   ```bash
   flutter run
   ```
